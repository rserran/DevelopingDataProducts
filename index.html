Shiny app description

  • The app uses the Longley economic dataset included in R.
  • The user selects the predictors (GNP.deflator, GNP, Armed Forces, Unemployed, Armed.Forces, Population, Year) to execute a multiple regression model to predict the Employed variable.
  • Once the predictors are selected, the outputs are:
    • Model coefficients, estimate standard errors, and p-values
    • Adjusted R square
    • Observer vs. Predicted plot
